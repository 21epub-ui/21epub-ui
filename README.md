# 21Epub Components

## Getting Started

```sh
corepack enable
yarn run bootstrap
```

## Usage

### Create a new package

```sh
yarn run create <package>
```

### Lint a package

```sh
yarn packages/<package> run lint
```

### Build a package

```sh
yarn packages/<package> run build
```

### Release a package

```sh
yarn packages/<package> run release <strategy>
```
